# LaserGRBL G-code 生成器 · 使用手册

适用版本：**v1.10.0** ｜ [English version](MANUAL.en.md) ｜ 本手册随每次版本发布同步更新，历史改动见 [CHANGELOG](../CHANGELOG.md)。

本工具是一个**单文件 HTML 应用**，面向机械臂绘画工作流：振动马达撒粉 + 激光燃烧双工具，也可作为普通激光雕刻/切割的 G-code 生成器使用。无需安装，浏览器打开 `gcode_generator.html` 即可（推荐 Chrome/Edge）。

---

## 1. 快速上手

1. 双击打开 `gcode_generator.html`；
2. 在顶部选择**作业模式**（激光 / 振动马达 / 双工序）；
3. 把 SVG 文件拖进导入区；
4. 在"图层参数"里为每个颜色图层设参；
5. 等待预览刷新（复杂图形在后台计算，界面不会卡）；
6. 点"下载"得到 `.nc` 文件，喂给 GRBL 即可。

**一个最简单的例子**：在 Illustrator/Inkscape 里画一个黑色描边的圆，导出 SVG，拖入工具 → 自动识别为一个黑色图层 → 路径模式选"描边" → 功率 30%、速度 1800 → 下载。生成的 G-code 会沿圆周走一圈。

---

## 2. 导入文件（SVG）

支持 path/line/polyline/polygon/rect/circle/ellipse 等元素。**图层按描边颜色自动划分**：图里用了几种颜色，就得到几个可独立设参的图层。颜色识别支持所有主流写法（CSS 类、内联 style、父级 `<g>` 继承、直接属性），Illustrator / Figma / Inkscape 导出的文件都能正确分层。

> 示例：想让"先撒粉的笔画"和"只用激光的细节"分开控制，在设计软件里把它们画成两种颜色（如红色和黑色）即可，导入后就是两个独立图层。

---

## 2.5 参数记忆与工程文件

**自动记忆**：所有全局设置和每个颜色图层的参数（功率/速度/模式/笔画风格等）自动保存在浏览器本地。刷新页面、明天再开都会原样恢复；重新导入使用相同颜色的 SVG 时，每个颜色图层自动套用上次的参数——"红色 = 构造线 S550 F600"这类映射一次设置、长期有效。

**工程文件**：导入区的"保存工程"把 **SVG 图形 + 全部参数（含撒粉档案）** 打包成一个 `.json` 下载；"载入工程"一键完整恢复。不同作品/参数方案各存一个文件，可长期归档、跨设备迁移、进 Git 管理。注意浏览器本地记忆在清除浏览器数据时会丢失，重要参数请存工程文件。

## 3. 全局设置

| 设置 | 说明 |
|---|---|
| 输出宽度 (mm) | 成品的实际宽度，高度按原图比例自动计算 |
| 原点 | 左下角（G-code 坐标全为正）或居中（原点在图形中心） |
| S 上限 ($30) | 必须与 GRBL 的 `$30` 设置一致，否则功率百分比不准。LaserGRBL 默认 1000 |
| 空程 G0 | 抬笔快速移动的速度，用于时间估算 |
| 气泵/风扇 (M8/M9) | 勾选后在开头/结尾插入 M8/M9 |
| 加工前走边框对位 | 正式加工前以零功率沿外框走一圈，确认位置和幅面 |
| 起始处置零 (G92 X0 Y0) | 把机械臂**当前停的位置**定义为本次作业的零点。起点不固定的大幅面场景必开 |
| 马达偏移 dX/dY | 撒粉喷口相对激光光点的物理偏移，见 §9.6 |
| 撒粉前预冲 (purge) | 正式撒粉前先到指定位置空撒几秒唤醒粉流，见 §9.5 |

## 4. 路径模式（激光图层）

每个图层可独立选择：

- **描边（outline）**：沿原始轮廓走一遍。适合切割和线稿。
- **中线（centerline）**：对闭合的粗笔画形状提取骨架中心线，只走一遍。适合把描边字体转成单线雕刻。
- **填充（fill）**：闭合形状内部用平行线扫描填满。参数见 §5。
- **加粗（thicken）**：把单线路径向两侧扩展成有宽度的笔画并填充。适合单线字体/手写路径。参数见 §6。

---

## 5. 填充参数

- **间距 (mm)**：相邻扫描线的间隔。表面雕刻常用 0.1~0.2；想刻透厚材料用 0.05 并配合多遍。
- **线迹方向**：
  - **固定角度**：传统方式，所有线一个方向。
  - **垂直笔画（法线）**：每条线垂直于笔画走向，长度≈笔画宽度。**解决厚材料上"有的地方刻得透有的刻不透"的问题**——长线全速巡航能量低、短线因加减速能量高，固定角度下不同朝向的笔画能量不均；垂直笔画让所有线迹等长，能量处处一致。转角处有角平分线分区 + 外伸补线，交叉处有自动去重，保证零空缺、零重叠。
- **兜底**：对"胖"形状（非笔画状，如实心圆盘）骨架提取可能为空，此时自动回退为固定角度填充。
- **填充+描边**：勾选后填充完再沿轮廓描一圈，边缘更整齐。

> 示例配方（厚卡纸刻字，之前刻不匀的场景）：填充间距 0.05、线迹方向选"垂直笔画"、S30 F1800、雕刻模式（M4），并确认 GRBL `$32=1`。

## 6. 加粗模式与笔画风格

**笔画宽度**滑块设定名义宽度（mm）。**笔画风格**决定宽度沿笔画如何变化：

| 风格 | 效果 | 适用 |
|---|---|---|
| 均匀 | 全程等宽 | 标准粗线 |
| 两端收锋 | 起笔收笔平滑收细到 20%，像钢笔入锋出锋 | 手写感 |
| 书法笔锋(斜切) | 宽度随笔画方向变化（横细竖粗），**笔锋角度**可调：30° 是西文斜体经典角度，0° 接近宋体笔意 | 书法字 |
| 毛笔 | 收锋 + 笔锋叠加 | 签名、题款 |
| 火药爆破 | 宽度噪声起伏 + 边缘灼烧短刺 + 周围飞溅火花，随机为确定性种子（同文件每次生成一致） | 爆炸/火药美学 |

加粗的填充同样支持"垂直笔画"线迹方向，且短线长度会跟随局部宽度——收锋处短线自动变短，能量依然均匀。闭合路径没有起止点，收锋自动退化为均匀（毛笔退化为纯笔锋）。

## 7. 作业模式

界面顶部第 0 节选择本次作业的模式，界面只显示该模式相关的设置：

| 模式 | 行为 |
|---|---|
| **激光雕刻** | 所有图层为激光；马达偏移、purge 等撒粉设置隐藏；材料预设可用 |
| **振动马达撒粉** | 所有图层为撒粉；材料预设隐藏（那是激光材料的概念）；马达偏移隐藏（单工具用 G92 置零即可）；purge 可用 |
| **双工序（先撒粉后激光）** | 每个图层可单独选设备；全部撒粉/双工具设置可见；输出两工序 G-code，中间 M0 换工具 |

模式选择会记住，下次打开保持上次的选择。

## 8. 设备与双工序（机械臂 撒粉+激光）

每个图层顶部的**设备**选项：**激光** 或 **振动马达（撒粉）**。两者共用一路 PWM，G-code 按物理换工具的流程分两个工序生成：

```
G92 X0 Y0            ← 可选：当前位置置零
(purge 预冲)          ← 可选
; PHASE 1 / POWDER   ← 全部撒粉图层（M3 恒定模式，坐标含马达偏移）
M0                   ← 暂停：换上激光头，按继续
; PHASE 2 / LASER    ← 全部激光图层（坐标无偏移）
```

撒粉图层永远排在激光之前，与 order 数值无关。只有单一设备时不产生 M0。

## 9. 撒粉图层详解

### 9.1 工作原理

激光是"细笔填色"（来回扫描填面积），撒粉是"粗笔一笔过"——粉线的宽度由振幅直接控制。所以马达图层**只走中线一遍**（开放路径原样走；闭合形状先提取骨架），笔画的粗细节奏全部编码在沿途逐段变化的 S 值（=振幅）里。撒粉工序用 **M3 恒定模式**而非 M4，避免加减速时振幅跟着掉、粉流不稳。

### 9.2 宽度节奏 → 振幅

选择笔画风格（均匀/收锋/笔锋/毛笔）决定设计上的粗细节奏，然后做**归一化映射**：设计中最细处 → 可用振幅下限，最粗处 → 上限，中间插值。这样即使设计里宽度差异很小，也会被拉伸成硬件能表现的最大反差。

- **最小/最大振幅 %**：无档案时的线性映射范围；
- **对比度 γ**：反差夸张程度。γ>1 让细的更细（笔锋更尖），γ<1 更饱满。默认 1；
- **前瞻补偿 (mm)**：粉从振动到落纸有延迟，此参数让振幅变化**提前**该距离发出，抵消滞后。笔画末端拖粉就加大它（从 3mm 试起）。

### 9.3 组件档案

每套"3D 打印撒粉件 + 粉料"组合的物理特性不同（出粉粗细、死区、饱和点）。**组件档案**保存每套组合的实测"振幅→线宽"曲线，图层里切换档案即自动套用对应映射。选"线性映射"则不用曲线、直接用振幅范围。

### 9.4 校准向导

马达图层点"校准向导…"：

1. **生成测试图案 G-code** → 下载 `calib.nc`（8 条阶梯振幅直线，20%~90%，每条 30mm）；
2. 装粉运行；
3. 用卡尺量每条粉线的实际宽度，填进表格（**没出粉的行填 0**，自动视为死区）；
4. 保存档案。

> 示例校准结果：30%→1.2mm、50%→2.0mm、70%→3.1mm、80%→3.5mm、20% 填 0（死区）。之后该档案下振幅永远只在 30~80% 这个有效区间里取值。

**快速漂移校准**（每天开工用）：粉量/湿度会让出粉量漂移。用基准振幅（如 60%）画一条线，量宽度填入"今日实测"，点缩放按钮——整条曲线按比例修正，十秒完成，不用重跑全套。

### 9.5 Purge 预冲

停机后管路里的粉是"睡着"的，直接开画第一笔会残。勾选 purge 后正式路径前自动到指定坐标（选画面外的废料区）空撒指定秒数唤醒粉流。

### 9.6 马达偏移标定

喷口和激光光点物理位置不同，必须标定：低功率激光画一个小十字 → 换撒粉头在**同一坐标**撒一个十字 → 量两个十字中心的偏差 → 填入 dX/dY。之后撒粉工序坐标自动平移，粉线与激光走纸上同一条线。工具头不拆装就不用重测。

## 10. 材料预设

"材料预设"下拉可一键套用 材质/厚度/效果 对应的 模式/功率/速度/遍数。预设管理器支持新建、从当前图层保存、删除、导入导出 JSON、恢复内置库。数据存在浏览器本地。

---

## 11. 预览与统计

- 滚轮缩放、拖拽平移、双击复位；
- 虚线为空程路径（超过 8000 条路径时自动隐藏以保流畅）；
- 马达图层以**变宽线**绘制，直接看到粉线粗细节奏；
- 悬停图层卡片高亮该层；
- 统计栏显示：优化前/后空程、节省比例、加工长度、预计时间、G-code 行数。

**性能说明**：所有重计算在后台线程执行，拖参数不卡界面；连续修改时只算最新一次。只改功率/速度/遍数等不影响几何的参数时命中缓存、瞬间出结果。超长 G-code 在文本框里截断显示，**下载的文件永远是完整的**。

## 12. G-code 结构参考

```gcode
; Generated by LaserGRBL G-code Generator (Fono) v1.8.0
G21            ; 毫米
G90            ; 绝对坐标
G92 X0 Y0      ; (可选) 当前位置置零
M8             ; (可选) 气泵
; --- framing ---            (可选) 零功率走边框
; ============ PHASE 1 / POWDER ============
G0 X-10 Y-10 S0
M3 S600                      ; purge 预冲
G4 P2.0
S0
M5
; ===== 图层名 · POWDER =====
M3
G0 X1.500 Y-2.000 S0         ; 坐标含马达偏移
G1 X2.480 Y-1.900 S312 F1500 ; S 逐段变化 = 振幅节奏
...
M5
M0 ; PAUSE: swap to LASER    ; 换工具，按继续
; ============ PHASE 2 / LASER ============
M4                           ; 雕刻动态模式 (切割为 M3)
G0 X0.000 Y10.000 S0
G1 X20.000 Y10.000 S300 F1800
...
M5
G0 X0.000 Y0.000 ; home
```

## 13. 常见问题

**导入 SVG 只有一个黑色图层？** 旧版本问题，v1.1 起已支持所有颜色写法。仍出现请检查 SVG 是否真的只用了一种描边色。

**厚材料有的地方刻不透？** 见 §5：填充线迹方向改"垂直笔画"，确认 `$32=1` 且雕刻用 M4。仍不透就加遍数而非无限降速。

**垂直笔画填充结果是空的？** 图形是"胖"形状，已自动回退固定角度（正常现象）。

**撒粉笔画末端有一坨粉？** 加大前瞻补偿（§9.2）。

**第一条粉线是残的？** 开启 purge 预冲（§9.5）。

**粉和激光烧的位置对不上？** 标定马达偏移（§9.6）。

**换了粉/换了打印件效果全变？** 每套组合建一个组件档案（§9.3），日常漂移用快速校准（§9.4）。

**火药爆破每次生成的火花不一样？** 不会——随机使用确定性种子，同一文件每次生成的 G-code 完全一致。
